## プロジェクト概要（1分で読む）

Adctorは、AIクローラーが無断でWebサイトのコンテンツを収集する問題を解決するための**AIアクセス課金インフラ**です。

```
AIクローラー → Cloudflare Edge → HTTP 402 → Stripe決済 → コンテンツ取得
                    ↑
            （人間は通常通り閲覧）
```

**ビジネスモデル:**
- サイトオーナーが導入 → AIクローラーに1アクセスあたり¥1〜¥100を課金
- Adctorが20%の手数料を徴収 → サイトオーナーに80%を還元
- サイトオーナーにとってゼロリスクで収益が生まれる

**なぜ今か:**
- Cloudflareが2025年に「Pay-per-Crawl」を公式機能として発表
- GPTBot、ClaudeBot、PerplexityBotなど主要AIが自動決済プロトコルに対応予定
- 日本市場での先行者優位を確立するタイミング
